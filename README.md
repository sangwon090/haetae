# Haetae

Yet another programming language, built for heterogeneous computing.  
프로그래밍 언어, 근데 이제 이기종 컴퓨팅을 곁들인.

## Key Features
- **Seamless Beyond-CPU Execution**  
  Effortlessly offload execution across CPUs, GPUs, and custom accelerators.

- **Pipeline-First, Immutable Paradigm**  
  Safe and predictable parallelism by design

- **Concise Syntax, Zero-Cost Abstractions**  
  High-level code compiled to efficient native kernels
